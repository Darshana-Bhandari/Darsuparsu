# NoteVault Pro

A modern and responsive Notes Management Application built with HTML, CSS, and JavaScript.

NoteVault Pro helps users create, organize, search, edit, and manage notes efficiently through a clean and intuitive interface. The application stores data locally using the browser's Local Storage API, ensuring notes remain available even after refreshing the page.

---

## Features

### Notes Management

* Create notes
* Edit notes
* Delete notes
* View note details
* Auto-save note drafts

### Organization

* Categorize notes

  * Personal
  * Work
  * Travel
  * Health
* Pin important notes
* Mark notes as favorites

### Search & Filtering

* Real-time search
* Category-based filtering
* Favorites filtering

### User Experience

* Glassmorphism-inspired UI
* Light Theme
* Dark Theme
* Responsive Layout
* Floating Action Button
* Smooth Animations
* Toast Notifications

### Dashboard Statistics

* Total Notes
* Pinned Notes
* Personal Notes Count
* Work Notes Count
* Recent Activity Tracking
* Live Clock

### Data Persistence

* Browser Local Storage
* Draft Recovery
* Export Notes as JSON

### Keyboard Shortcuts

| Shortcut | Function           |
| -------- | ------------------ |
| Ctrl + N | Create New Note    |
| Ctrl + S | Save Note          |
| Esc      | Close Active Modal |

Clone the repository:

Open the project folder:

```bash

```

Run the application by opening `index.html` in your browser.
